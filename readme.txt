Plugin de conexão com o sistema da [InCuca Tech](https://app.incuca.net/).

== Descrição ==

Este plugin é uma maneira prática de conectar o seu WordPress ao sistema da [InCuca Tech](https://app.incuca.net/).

O plugin faz a conexão com o sistema via webhook e integra o seu website através de um token único. Assim, você consegue usufruir de todas as funcionalidades que a integração habilita.

Após conectado, ele também permite você adicionar o pixel de rastreamento com um clique. O pixel irá coletar dados de visitantes conforme você configurar no sistema.

== Instalação ==

1. Para instalar e ativar o plugin conector da InCuca, há três maneiras:

  - Baixar e descompactar o arquivo `*.zip` na pasta  `/wp-content/plugins/` ;
  - Fazer o upload do arquivo `*.zip`  via plugins do WordPress em  **Plugins &gt; Adicionar plugin &gt; Enviar plugin**
  - Instalar e ativar o plugin por meio do menu **Plugins** no WordPress

2. Vá para  **Configurações &gt; InCuca conexão** e insera o seu código.
